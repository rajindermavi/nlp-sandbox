# nlp-sandbox


### Start Services:
Some notebooks utilize Elastic Search and Qdrant, the following commands deploy docker containers to run these services.

* Start Elastic Search:
```
make es-up
```
* Stop Elastic Search
```
make es-down
```

* Start Qdrant:
```
make qdrant-up
```
* Stop Qdrant
```
make qdrant-down
```
